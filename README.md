# Alzheimer's-Detection-AI-Tool

[cite_start]A multi-modal diagnostic pipeline using **Whisper**, **Visual Language Models (VLM)**, and **Llama (LLM)** to detect Alzheimer’s disease through automated speech and visual cue analysis[cite: 25, 26, 27].

## Key Technical Features
* [cite_start]**Speech Processing:** Utilizes **Whisper** for high-accuracy speech-to-text transcription of patient audio recordings[cite: 26].
* [cite_start]**Visual Cues:** Employs **Visual Language Models (VLM)** to extract and analyze non-verbal cognitive indicators from patient data[cite: 26].
* [cite_start]**Reasoning Engine:** Implements a **Chain-of-Thought (CoT)** prompting strategy to guide the diagnostic process through logical steps[cite: 27].
* [cite_start]**LLM Integration:** Leverages **Llama (LLM)** to correlate transcripts and visual data for an automated assessment of cognitive decline[cite: 27].

## Methodology
* [cite_start]**Data Fusion:** Processes audio and text data simultaneously to identify complex patterns associated with cognitive impairment[cite: 28].
* [cite_start]**Diagnostic Optimization:** Focused on identifying specific linguistic and visual markers to improve classification accuracy and automated patient assessment[cite: 28].

## Technologies Used
* [cite_start]**Models:** Whisper (OpenAI), Llama (Meta), and Visual Language Models (VLM)[cite: 26, 27].
* [cite_start]**Platform:** Developed and tested using **Kaggle** environments[cite: 45].
* [cite_start]**Languages:** Python[cite: 37, 45].
