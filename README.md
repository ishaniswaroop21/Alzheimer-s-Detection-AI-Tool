# Alzheimer's-Detection-AI-Tool

A multi-modal diagnostic pipeline using **Whisper**, **Visual Language Models (VLM)**, and **Llama (LLM)** to detect Alzheimer’s disease through automated speech and visual cue analysis.

## Key Technical Features
* **Speech Processing:** Utilizes **Whisper** for high-accuracy speech-to-text transcription of patient audio recordings.
* **Visual Cues:** Employs **Visual Language Models (VLM)** to extract and analyze non-verbal cognitive indicators from patient data.
* **Reasoning Engine:** Implements a **Chain-of-Thought (CoT)** prompting strategy to guide the diagnostic process through logical steps.
* **LLM Integration:** Leverages **Llama (LLM)** to correlate transcripts and visual data for an automated assessment of cognitive decline.

## Methodology
* **Data Fusion:** Processes audio and text data simultaneously to identify complex patterns associated with cognitive impairment.
* **Diagnostic Optimization:** Focused on identifying specific linguistic and visual markers to improve classification accuracy and automated patient assessment.

## Technologies Used
* **Models:** Whisper (OpenAI), Llama (Meta), and Visual Language Models (VLM).
* **Platform:** Developed and tested using **Kaggle** environments.
* **Languages:** Python.
